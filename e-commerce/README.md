#  ShopyKura – E-Commerce Website (React)

ShopyKura is a modern, responsive e-commerce website built using **React** as part of my internship task.  
The project demonstrates real-world shopping features like product listing, add to cart, checkout flow, and multiple pages with a professional UI.



## Features

- Responsive & professional UI (Green & White theme)
- Home page with long hero section and multiple content blocks
- Products page with 20 products (uniform image size)
- Add to Cart with confirmation alert
- Cart icon updates when products are added
- Checkout page with total price calculation
- Why Us page with detailed content and hero section
- Contact page with hero section, left details & right contact form
- Sticky header and long footer
- State management using React Context API
- Routing using React Router

---

##  Tech Stack

- React
- React Router DOM
- Context API
- HTML5
- CSS3 (Flexbox & Grid)
